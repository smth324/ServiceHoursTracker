Hello, this is a sample python django project for Scholars Service Hours Tracking. Scholars
can view service hours opportunities, track how many they need, generate QR codes for verification
and automatically filter which ones fit their schedule. Organizations can also use this to request
service hours from scholars for their events and easily verify their information and attendance by scanning
the QR Code. The Office of Admissions can also view all the information on their dashboard and see which
students are lacking.

## To run the project locally:
1. pip install qrcode[pil]
2. Run "python manage.py runserver"
